## 🎯 What is ProofHire?

ProofHire is an AI-powered talent screening tool that helps recruiters:

1. **Screen faster** — Gemini AI analyzes all candidates simultaneously and ranks them
2. **Screen fairer** — Built with African context: a dev from Kigali is evaluated as fairly as one from San Francisco
3. **Screen smarter** — Candidates verify their skills with a 5-minute AI-generated test before reaching the recruiter

---

## 🚀 Key Features

### For Recruiters
- 📋 **Job Management** — Create jobs with required skills and experience level
- 📥 **Applicant Ingestion** — Import via structured JSON profiles or CSV upload
- ⚡ **One-Click AI Screening** — Gemini AI screens all candidates and returns a ranked Top 10/20 shortlist
- 🧠 **Full AI Transparency** — Every candidate gets a match score, strengths, gaps, and natural language reasoning
- ✅ **Verified Badges** — Instantly spot candidates who passed the skill verification test

### For Candidates
- 🛡️ **Skill Verification Test** — 5-minute personalized test generated by Gemini based on declared skills
- 🏆 **Verified Profile Badge** — Pass 70%+ and your profile stands out from the crowd
- 🌍 **Fair Evaluation** — African context is respected: hackathon wins, local projects, and real-world impact count

## 🤖 AI Decision Flow

```text
1. Recruiter triggers screening for a job
        ↓
2. Backend fetches all applicants for that job from MongoDB
        ↓
3. All profiles compiled into ONE structured Gemini prompt
   → System instruction: "Judge based on skills, not company names.
     African context is valued. Hackathon wins count."
        ↓
4. Gemini 1.5 Flash evaluates ALL candidates simultaneously
   → Returns ranked JSON: score, strengths, gaps, recommendation
        ↓
5. Results saved to MongoDB, returned to recruiter dashboard
        ↓
6. Candidate takes a 5-min skill test (questions generated by Gemini)
   → correctAnswer never leaves the server
   → 70%+ = isVerified: true in MongoDB = Verified badge on profile
```

## 🛠️ Local Setup

### Prerequisites
- Node.js 18+
- MongoDB Atlas account (free tier)
- Google AI Studio API key (free at aistudio.google.com)

### 1. Clone the repos
```bash
git clone https://github.com/your-team/proofhire-backend
git clone https://github.com/your-team/proofhire-frontend
```

### 2. Setup Backend
```bash
cd proofhire-backend
npm install

# Create .env file
cp .env.example .env
# Edit .env with your credentials:
# PORT=5000
# MONGODB_URI=mongodb+srv://...
# GEMINI_API_KEY=your_key_here
# FRONTEND_URL=http://localhost:3000

# Seed the database with test data
npm run seed

# Start dev server
npm run dev
# → Backend running on http://localhost:5000
```

### 3. Setup Frontend
```bash
cd proofhire-frontend
npm install

# Create .env.local file
echo "NEXT_PUBLIC_API_URL=http://localhost:5000" > .env.local

# Start dev server
npm run dev
# → Frontend running on http://localhost:3000
```

## ⚠️ Assumptions & Limitations

- **Gemini rate limits**: Screening large batches (50+ candidates) may hit API rate limits. Current implementation processes all candidates in a single prompt for speed.
- **CSV import**: Supports basic profile fields only (firstName, lastName, email, headline, location, skills). Full nested experience/education requires JSON profile format.
- **Test security**: Correct answers are stored server-side only and never exposed to the client. However, the system is not designed to prevent screen sharing during tests.
- **AI fairness**: Our Gemini prompt is engineered for African context, but AI scoring should always be reviewed by a human recruiter before making final decisions.
- **Free tier limits**: MongoDB Atlas M0 supports 512MB storage. Railway free tier supports 500 hours/month execution time.
